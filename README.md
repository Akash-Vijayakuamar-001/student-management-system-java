# Student Management System (Java + JDBC + MySQL)

A console-based Student Management System built using **Java**, **JDBC**, and **MySQL** to manage student records through CRUD operations.

## Features
- Add Student
- View All Students
- Search Student by ID
- Update Student
- Delete Student
- Clean console menu-based workflow

## Tech Stack
- Java
- JDBC
- MySQL
- SQL

## Concepts Used
- OOP (Model class)
- DAO Pattern
- Exception Handling
- Input Validation
- Modular & Clean Code

## Database Setup
1. Install MySQL
2. Create database and table using `database.sql`

## How to Run
1. Update DB username/password in `src/DBConnection.java`
2. Compile and run `src/Main.java`

## Sample Menu
1. Add Student
2. View All Students
3. Search Student by ID
4. Update Student
5. Delete Student
6. Exit
